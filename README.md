# AI Customer Support Agent

## Overview

An AI-powered customer support agent built using:

- OpenAI GPT
- LangChain
- LangGraph
- Pinecone Vector Database
- Streamlit
- Google Colab

## Features

- Product search
- RAG-based question answering
- Alternative product recommendation
- Customer support workflows
- Conversation memory
- Source citation

## Folder Structure

documents/      -> PDFs
logs/           -> Chat logs
screenshots/    -> Project screenshots
reports/        -> Final report
